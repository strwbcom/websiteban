# Ban Websites
An extension that blocks websites with the TikTok ban popup.


## Installation

### 1. Download the extension
Click the green "Code" button, download as a ZIP, and unzip it.

### 2. Open Chrome Extensions
1. Open Google Chrome.
2. Navigate to `chrome://extensions/` in the address bar.
3. Enable **Developer mode** (toggle it on in the top-right corner).

### 3. Load the Extension
1. Click **Load unpacked**.
2. Select the folder where you cloned/download the repository.
3. The extension will now appear in your list of installed extensions.

### 4. Add Websites to the List
Click the extension, add the websites you want to block, and save the changes.

### 5. Enjoy the bans.
